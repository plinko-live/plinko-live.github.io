How this Plinko demo works

This is a simplified browser version of a Plinko-style game. Each slot at the bottom has its own multiplier. When you press the button, the ball drops into a random slot, and your potential win is calculated as bet × multiplier.

The goal of this demo is to show how a basic game can be built with plain HTML, CSS and JavaScript and hosted on GitHub Pages without any backend.

This demo is for educational purposes only and does not represent a real-money gambling product.
